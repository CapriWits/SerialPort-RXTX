## Feature

- `ARM & C` initialize hardware
- Using `Java RXTX` achieve Serial Ports commnication
- Using `EasyExcel`  to serialize ArrayList into Excel



## How To Use

1. Move `.dll` files to `JRE\bin` 
2. Maven build project
